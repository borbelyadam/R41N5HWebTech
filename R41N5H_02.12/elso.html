GitHub Parancsok

Local Parancsok

        
            git init                  # Kezdeti Git inicializálás a helyi repository létrehozásához
            git add        # Állomány hozzáadása a staging area-hoz
            git commit -m "Message"  # Staging area tartalmának commitolása
            git status               # Repository állapotának ellenőrzése
            git log                  # Commitok naplózása

Remote Parancsok

        
            git remote add origin   # Távoli repository hozzáadása
            git push -u origin master               # Lokális változások feltöltése a távoli repository-ba
            git pull origin master                  # Távoli változások letöltése a lokális repository-ba
